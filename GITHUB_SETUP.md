# The Squishery — GitHub Setup Guide

---

## Transfer the repository to a new account

1. Go to **github.com/finops-evg/thesquishery.shop**
2. Click **Settings** (top tab)
3. Scroll to the bottom — find the **Danger Zone** section
4. Click **Transfer repository**
5. Type the new GitHub username
6. Confirm by typing the repo name when prompted
7. Click **I understand, transfer this repository**

The new owner receives an email — once they accept, the repo moves to their account and the website continues working automatically.

---

## Give yourself collaborator access after the transfer

Ask the new owner to do this:

1. Go to **github.com/NEWUSERNAME/thesquishery.shop**
2. Click **Settings → Collaborators → Add people**
3. Type your GitHub username and click **Add**
4. You accept the invite via email

Once accepted, you can push updates to the site the same way as before.
